\# Gestor de Notas – Evaluación 3



Este proyecto es un programa en Python que permite gestionar notas de alumnos mediante un menú interactivo. El usuario puede agregar notas, listarlas, calcular el promedio general, ver cuántas notas se han registrado y conocer la nota más alta.



\## Características del programa



\- Uso de \*\*funciones\*\* para organizar el código.  

\- Uso de \*\*listas y diccionarios\*\* para almacenar las notas.  

\- Validación de entradas del usuario con manejo de errores.  

\- Menú interactivo con opciones numeradas.  

\- Implementación de un ciclo \*\*while\*\* para mantener el programa en ejecución.  

\- Mejoras adicionales añadidas en una rama separada (\*mejora\*):

&nbsp; - Mensaje visual al finalizar el programa.



\## Cómo ejecutar el programa



1\. Abrir una terminal en la carpeta del proyecto.  

2\. Ejecutar el siguiente comando:



```

python prueba.py

```



3\. Seleccionar una opción del menú escribiendo un número del 1 al 6.



\## Requisitos



\- Python 3.10 o superior.

\- Sistema operativo Windows, Linux o MacOS.



\## Uso de Git



Durante el desarrollo de este proyecto se realizaron los siguientes pasos:



\- Inicialización de un repositorio Git.  

\- Realización de \*\*commits significativos\*\* en la rama \*master\*.  

\- Creación de la rama \*mejora\* para implementar una funcionalidad adicional.  

\- Fusión (\*merge\*) de la mejora hacia la rama principal.
